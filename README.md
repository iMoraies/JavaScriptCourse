# JavaScriptCourse

Repositório com meus estudos de **JavaScript**, criado para organizar e praticar os conteúdos aprendidos nas minhas aulas.

Este projeto contém várias pastas com exercícios e exemplos de código em HTML, CSS e JavaScript, para acompanhar meu aprendizado desde os primeiros conceitos até práticas mais avançadas.

---

## Estrutura das aulas

O repositório está organizado por pastas de aula com os arquivos correspondentes:

**Exemplos incluídos até agora:**
- `Aula 01` – Alert, confirm e prompt
- `Aula 02` – Conceitos e práticas intermediárias
- `Aula 03` – Continuação e conceitos extras
- `Practice/Excercice 01` – exercícios práticos
